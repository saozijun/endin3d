# 项目介绍

这个项目是一个使用 Vue.js 应用程序，创建 3D 天线设备信息建模 可视化的应用。它旨在为用户提供一个交互式和沉浸式的体验，以探索和分析设备。

## 特点

- 3D 可视化：该应用程序在 Web 浏览器中渲染 3D 模型并显示它们。
- BIM 集成：支持导入 glb 模型，通过 CAD 图纸，以生成 3D 模型。
- 实时天线数据: 提供实时天线，主机实时状态，快速洞察提前抑制所有风险。
- 交互控制：用户可以使用直观的控件（如平移、缩放和旋转）来浏览和与 3D 场景交互。
- 注释：在 3D 模型中的特定对象或区域提供额外的信息或上下文。
- 剖面视图：提供创建 3D 模型的剖面视图的功能，允许用户可视化地铁站点的内部结构。
- 照明和着色：该应用程序支持逼真的照明和着色效果，以提高 3D 场景的视觉质量。
- 响应式设计：设计为响应式并兼容不同的屏幕尺寸，使其适用于桌面和移动设备。

## 安装

要在本地运行此项目，请按照以下步骤操作：

1. 进入项目目录：`cd vue-project`
2. 安装依赖项：`npm install`
3. 启动开发服务器：`npm run dev`
4. 项目打包: `npm run build`, 即可在dist文件夹中查看打包生成的文件

## 使用

一旦应用程序运行起来，您可以执行以下操作：

- 使用鼠标或触摸手势浏览 3D 场景。
- 单击对象以查看更多信息。
- 使用提供的控件来操作视图，如缩放和旋转。

## 模型信息

### 模型命名规范

目的：有规律的命名可使得程序自动渲染

1. 耦合器名称列表: OH-1...100 例如 OH-1 OH-2 OH-3...OH-100
2. 功分器名称列表: GF-1...100 例如 GF-1 GF-2 GF-3...GF-100
3. 墙面高光对象名称： JZ-QT-1...100 例如 JZ-QT-1 JZ-QT-2 JZ-QT-3 ... JZ-QT-100
4. 电梯扶手表面名称列表： JZ-FT--fs-1...100 例如 JZ-FT--fs-1 JZ-FT--fs-2 JZ-FT--fs-3...JZ-FT--fs-100
5. 电梯表面名称列表： JZ-FT-1...100 例如 JZ-FT-1 JZ-FT-2 JZ-FT-3...JZ-FT-100
6. 馈线名称列表：KX
7. 天线命名：根据上面 Excel 表

### 天线规范

1. 模型的天线大小：要和 下应南站 大小一致
2. 模型的天线朝向： 要和下应南站 朝向 保持一致

## 集成信息

通过 iframe 的方式进行集成模型引擎，其中模型引擎的数据来源于业务方接口

### 集成须知

1. iframe 需要设置宽高
2. token（eleToken）获取 token 信息，
   站点信息（stationName）
   站点 ID（stationId）
   站点接口请求频率（intervalTime）默认是 15 秒，
   都通过 localStore 的方式与模型引擎进行交互。
3. 站点模型的切换是通过站点 ID 与模型同名文件进行绑定，动态切换。
4. 站点数据模拟通过 iframe src 设置 ?isTrue=true 开启。 删除 isTrue 进行关闭。例如
   开启：http://127.0.0.1:5500/dist/index.html?isTest=true
   关闭： http://127.0.0.1:5500/dist/index.html
