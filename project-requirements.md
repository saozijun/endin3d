# 项目需求说明

## 一、概述
提供一个可以集成的模块给`网管平台`（另一个vue工程），实现3D引擎，支持指定路径的glb、gltf格式模型加载，并对接指定的服务器，实现告警数据交互。

## 二、集成环境
集成环境相应依赖版本为：
```
"node":"12.18.3", 
"npm":"6.14.6",
```
## 三、功能需求

### 3.1 模型加载
支持glb、gltf格式模型加载，支持指定路径的模型加载。

### 3.2 环境贴图
支持环境贴图加载，支持指定路径的贴图加载。

### 3.3 模型渲染
支持模型渲染为蓝色半透明玻璃材质效果。

### 3.4 模型交互
支持模型交互，支持模型缩放、旋转、平移。

### 3.5 模型控制工具栏
提供模型控制工具栏，支持模型的视角重置、设备信息隐藏。

### 3.6 设备标注
读取模型中预设的设备信息，并标注在模型上。

### 3.7 设备告警
预设设备告警效果，绑定到设备标注上。

### 3.8 告警数据交互
对接指定的服务器，实现告警数据交互。

### 3.9 浏览器适配与兼容
支持不同分辨率PC浏览器的适配，支持Edge、Chrome、Firefox等主流浏览器。

### 3.10 集成页面
提供集成页面，提供iframe的方式被`网管平台`按需集成。

## 四、技术选型

### 4.1 工程基础框架
选用`vue-cli`, 路由选用`vue-router`, 状态管理选用`vuex`.

### 4.2 3D基础框架
选用`three.js`, 模型加载选用`GLTFLoader`, 环境贴图选用`HDRCubeTextureLoader`, 告警数据交互选用`socket.io`.

### 4.3 浏览器适配与兼容
选用`babel-polyfill`实现浏览器兼容。

### 4.4 CSS预处理
选用`sass`作为CSS预处理语言。
